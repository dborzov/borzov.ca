Open source projects

See my github account page for details.

lsp - ls shell command alternative reimagined for modern times, with friendlier notations, built-in awareness of things like git repos and various dotfiles file datatypes.

NetVis - a tool to visualize and understand networks. Translate your network logs to a simple JSON-based NetVis format and visualize the network events and communication in detail. Check out the demo. Built in coordination with IPFS team, NetVis is an indispensable tool when designing and developing network functionality on any level.

RusticSearch - is a simple search engine server. It uses in-memory suffix array to lookup index entries. RusticSearch is minimalistic in design and perfect when adding simple search functionality where fancier solutions can be an overkill.
